# docs

Documentation for features and formats that are shared across two or
more tools in this repo.

Tool-specific docs live alongside the tool they describe (e.g.,
`options-scanner/SCHWAB_DATA_SOURCE.md`). Anything here applies
repo-wide.

## Contents

| File | What it covers |
|------|---------------|
| [glossary.md](glossary.md) | Definitions of options and quantitative-finance terms used throughout the repo — single-leg basics (call/put, strike, ITM/ATM/OTM), the Greeks (delta/gamma/theta/vega/GEX), implied volatility, spread strategies (vertical, condor, calendar, straddle, etc.), and market-structure (NBBO, OI, volume) |
| [stockpile-format.md](stockpile-format.md) | Manual transaction log format accepted by the options scanner portfolio tab and the positions tracker |
